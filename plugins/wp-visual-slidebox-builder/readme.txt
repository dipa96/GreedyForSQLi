== Description ==

[youtube https://www.youtube.com/watch?v=_gwF-RcZzYU]

<h4>
Add cool hover effects to images or any other type of content.
</h4>
<p>
Build amazing hover animations without writing any code. Real time preview as you are building to guide you. Fully customizable! No coding experience is necessary, everything is done through
the UI, which is very intuitive and easy to use. Choose from libraries of built in animations, and builder types. Build simple hover effects or take it to the next level with our advanced features. Create grids of hover animations.
</p>

<h4>How to Use</h4>
<ul>
<li>Create your animation using the powerful UI builder</li>
<li>Save it and copy the shortcode generated by the plugin</li>
<li>Paste the shortcode anywhere on your site</li>
<li>Update your existing animations, changes propagate anywhere the animation is used </li>
</ul>


<h4>Features</h4>
<ul>
<li>Fully configurable</li>
<li>WYSIWYG editor - Live preview</li>
<li>Grid Builder allows you to create grids of hover animations in seconds</li>
<li>Clone your exciting creations to save time</li>
<li>Insert slide-box anywhere on your site using shortcode</li>
<li>Large pool of animations and effects</li>
<li>Edit a slide box and changes propagate automatically</li>
<li>History of all your slide box creation</li>
<li>Build smart using builder types, each tailored for specific needs</li>
</ul>

== Installation ==

1. Upload and extract the zip file downloaded to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. The plugin should appear in 'Settings'
