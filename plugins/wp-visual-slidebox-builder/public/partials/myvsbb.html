What's new?

We have given this plugin a make over! We are excited to present you with the new 3.0 version of Visual Animation Builder. Taking feedback from many users and personal experiences we have added new features, improved exciting features, upgraded the backend and frontend to more modern technologies. Here is a brief breakdown on what you can expect. For more details on features and more please visit our website wpvisualslideboxbuilder.com

New Features Learn More

• Builder Types

We have added multiple builder types to fit your needs. Some just need a quick and simple animation, while others might need a more custom and explicit animation. By adding builder types we are not limiting those with extensive customizations and at the same time we are not over complicating the process for those looking for something simple and quick.

• Grid Builder

We have added multiple builder types to fit your needs. Some just need a quick and simple animation, while others might need a more custom and explicit animation. By adding builder types we are not limiting those with extensive customizations and at the same time we are not over complicating the process for those looking for something simple and quick.

• Hover State Preview

This feature allows you to see the changes you are making to the hiden portion of the animation by making it appear on the screen while you edit it, so you don't have to manually trigger it each time by hovering over.

• Opacity/Transparency for all animation type

In the pass the opacity setting was only meant to be used with a handful of effects. We have changed how this feature is applied to allow it to be used in every single effect and transition.

• WYSIWYG Text Editor

We added WYSIWYG text editor to make your life easier.

Improved Features Learn More

• Responsive Layouts

We have enhanced and simplified the responsive feature. Now you can choose more options to compliment your responsive animation, such as gutters, whether or not you wish to preserve the ratio when resizing, and we added a more comprehensive layout with upto 6 columns.

• A lot more transitions and effects

We upgraded our transitions and effects list to include a lot more. Now you have over 38 effects and transitions to choose from.

• Updating/Upgrading

We simplified the process of updates and upgrades. For those looking to purchase a paid version, the process is a lot simpler now. Simply copy and paste the license and you are all set! No need to uninstall or install anything.

Legacy Support Learn More

What about legacy support?

We will continue to support the rendering of legacy animations. This means your site won't break if it is using legacy creations somewhere in its pages. However, we will no longer support editing legacy animations. You have access to your legacy animations on the history page, from where you can access the shortcodes and use it. However, we strongly encourage users to use the new builder types to improve performance and usability.
